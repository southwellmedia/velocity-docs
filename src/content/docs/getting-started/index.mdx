Get your Velocity project running in just a few commands.

## Using the CLI (Recommended)

The fastest way to start is with the official CLI:

<Tabs>
  <TabItem label="pnpm" icon="pnpm">
```bash
pnpm create velocity-astro my-project
cd my-project
pnpm dev
```
  </TabItem>
  <TabItem label="npm" icon="seti:npm">
```bash
npm create velocity-astro@latest my-project
cd my-project
npm run dev
```
  </TabItem>
  <TabItem label="yarn" icon="seti:yarn">
```bash
yarn create velocity-astro my-project
cd my-project
yarn dev
```
  </TabItem>
  <TabItem label="bun">
```bash
bun create velocity-astro my-project
cd my-project
bun run dev
```
  </TabItem>
</Tabs>

<Aside type="tip" title="CLI Features">
  The CLI lets you choose components, enable i18n, and customize your setup.
  [Learn more →](/cli/)
</Aside>

## Configure Environment

```bash
cp .env.example .env
```

Edit `.env` and set your production domain:

```bash
SITE_URL=https://yoursite.com
```

## Start Development

```bash
pnpm dev
```

Your site is now available at `http://localhost:4321`.

## Configuration Checklist

<Steps>
1. **Set your domain** in `.env` → `SITE_URL=https://yoursite.com`

2. **Update site info** in `src/config/site.config.ts` → name, description, author, contact

3. **Configure navigation** in `src/config/routes.ts` → add/remove pages from nav

4. **Replace branding** in `src/assets/branding/` → your logo SVGs

5. **Update favicon** in `public/favicon.svg` → your icon
</Steps>
